# Бот для планирования задач

Этот бот предоставляет функционал для управления задачами через Telegram.

## Установка

1. Установите необходимые зависимости, выполнив команду:
pip install -r requirements.txt

2. Внесите необходимые изменения:
- В файле `bot.py` установите значение соответствующей строки bot = telebot.TeleBot('Your Token') равным токену вашего бота Telegram.
- Убедитесь, что адрес базы данных PostgreSQL указан правильно в файле `database.py` и `alembic.ini` в соответствующих переменных.

3. Примените миграции базы данных:
alembic upgrade head


## Использование

Запустите скрипт `bot.py`, чтобы запустить бота. Теперь вы можете управлять задачами через Telegram, используя доступные команды.
